About the IBM DataPower Add-on for Splunk
=========================================

.. csv-table::
   :file: csv/about.csv

The IBM DataPower Add-on for Splunk integrates Remedy brings field extractions to Splunk platform. This add-on does not collect any data.

Download the `IBM DataPower Add-on for Splunk`_ from Splunkbase.

For a summary of new features, fixed issues, and known issues, see `Release Notes for the IBM DataPower Add-on for Splunk`_.

For information about installing and configuring the IBM DataPower Add-on for Splunk, see `Installation and configuration overview for the IBM DataPower Add-on for Splunk`_.
